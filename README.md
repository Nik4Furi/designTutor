## <h2 id="runLocally" >Run Locally </h2>

Clone the project

```bash
  git clone https://github/Nik4Furi/designTutor
```

Go to the project directory

```bash
  cd designTutor
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
